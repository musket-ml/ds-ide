Outline

How to show

If not already active please open Window->Show View->Outline.

What is contained?

You will find there the complete content of your editor file in hierarchical style.

How can I jump to text for entry shown in outline?

You must double click on a tree item

When is the tree updated?

Changes on your script are only visible inside outline view when you save the script. Unsaved changes will not appear.
